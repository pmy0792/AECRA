Backing Up

There are only two assets that you need to back up on your Dungbeetle server.

  • The database
  • The screenshots

If your server uses MySQL:

  • Use mysqldump to back up your database.
  • Back up Screenshots.

If your server uses SQLite:

  • Back up DungbeetleDatabase.sqlite.
  • Back up Screenshots.

Command line flags can be used to place the assets in specific places, so make sure you back up appropriately.

Run DungbeetleServer.exe --help for path details.

That should be it! You're all backed up.
