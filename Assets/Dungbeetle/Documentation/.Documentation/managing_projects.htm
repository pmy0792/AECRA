Managing projects

Dungbeetle servers allow different Unity projects to have separate lists of bugs, even while connecting to the same server.

You can change a Unity project's project id in the connection config window.

Project developers

A project developer is a name to which a bug report can be assigned in different ways.

In the Assigned tab in the overview window, bug reports are filtered by which developer is assigned as Fixer. This is the most important aspect of project developers.
